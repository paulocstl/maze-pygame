Maze Escape

Português

Descrição

Maze Escape é um jogo 2D desenvolvido em Python utilizando a biblioteca Pygame.

O objetivo do jogador é explorar o labirinto, encontrar a chave e chegar até a porta para vencer a partida. O jogo possui mapas aleatórios, sistema de colisão e reinício após a vitória.

Funcionalidades:
 -Movimentação com WASD.
 -Sistema de colisão com paredes.
 -Cinco mapas diferentes.
 -Seleção aleatória do mapa.
 -Coleta de chave.
 -Sistema de vitória.
 -Reinício da partida pressionando R.
 -Sprites personalizados
 
 Tecnologias utilizadas:
 -Python 3
 -Pygame

 
  # English #

Description

Maze Escape is a 2D game developed in Python using the Pygame library.

The player's objective is to explore the maze, find the key, and reach the door to win the game. The game includes random maps, collision detection, and a restart system after victory.

Features:
 -WASD movement.
 -Wall collision system.
 -Five different maps.
 -Random map selection.
 -Key collection system.
 -Victory condition.
 -Restart game by pressing R.
 -Custom sprites.

Technologies:
 -Python 3
 -Pygame
